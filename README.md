# bfjit

## Syntax

| Brainfuck | C         |
|:---------:|:---------:|
| >         | ptr++     |
| <         | ptr--     |
| +         | *ptr++    |
| -         | *ptr--    |
| .         | fputc(*ptr, stdout) |
| ,         | *ptr=fgetc(stdin)   |
| [         | while(*ptr) {       |
| ]         | }                   |
